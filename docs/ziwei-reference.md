# 倪海厦天纪 · 紫微斗数知识体系

> 来源：倪海厦《天纪》课程
> 用途：warmstart 项目紫微斗数参考文档

## 一、核心框架：命/运/地理三分法

命占三分之一，运（大运流年）占三分之一，地理（阳宅风水、人事选择）占三分之一。

"命是天注定的，运掌握在我们手上"——先天条件决定起点，后天选择和环境决定终点。
人占三分之一，地理占三分之二，又大于天命——环境调整的力量大于先天命定。
"命好不如限好"——倪师反复强调：大运流年的好坏，有时比本命格局更重要。
"祸福无门，惟人自召"——吉凶是人的选择和心念招来的，不是凭空来的。
批命需结合三方四正的互动关系，而非孤立解读单宫信息。

## 二、星曜体系

### 主星（14颗）
| 星曜 | 特质 | Agent 映射 |
|------|------|-----------|
| 紫微 | 帝星，尊贵统御，需辅弼配合化煞 | 领导型人格 |
| 天机 | 智谋变动，善机变，财从脑生 | 策略型思维 |
| 太阳 | 光明博爱发散，代表父亲/儿子/丈夫（女命） | 外向影响力 |
| 武曲 | 刚毅财星，执行力，主财权 | 执行型行动力 |
| 天同 | 温和福星，协调力 | 关系型亲和力 |
| 廉贞 | 复杂，桃花血光，情绪之火 | 情感强度 |
| 天府 | 南斗主星，解厄，稳重 | 稳健型判断 |
| 太阴 | 温柔，代表母亲/妻子（男命） | 内敛型共情 |
| 贪狼 | 欲望桃花，主才艺 | 探索欲 |
| 巨门 | 是非口舌，暗伏 | 批判性思维 |
| 天相 | 印星，辅佐协调 | 协作型 |
| 天梁 | 荫星，长辈缘，解厄 | 守护型 |
| 七杀 | 刚烈变动，武职 | 决断力 |
| 破军 | 破坏重组 | 变革力 |

### 杀破狼格局
七杀、破军、贪狼三颗星合称"杀破狼"。个性都很刚烈：
- 七杀：性格刚急
- 破军：破坏重组
- 贪狼：欲望才艺

### 四化星（核心推命工具）
- 科星：后发成名，名望考试升迁学识
- 权星：权力地位，管理能力魄力
- 禄星：稳定财路，财禄收入
- 忌星：意外破财与健康预警。"忌星所在宫位就是这一生最需要修行的领域"

### 四煞星
擎羊（冲突突发）、陀罗（拖延慢性）、火星（暴躁急性）、铃星（暗中是非）

## 三、十二宫体系

| 宫位 | 含义 | Agent 画像映射 |
|------|------|---------------|
| 命宫 | 先天性格和命运格局 | 人格基调 |
| 兄弟宫 | 手足关系，平辈竞争合作 | 合作风格 |
| 夫妻宫 | 婚姻质量、感情走向 | 亲密关系模式 |
| 子女宫 | 子女缘分、桃花 | 创造性产出 |
| 财帛宫 | 财运、赚钱方式 | 价值偏好 |
| 疾厄宫 | 健康隐患 | 注意力低谷/疲劳信号 |
| 迁移宫 | 外出发展、人际关系 | 对外表现模式 |
| 交友宫 | 朋友下属合作伙伴 | 社交策略 |
| 官禄宫 | 事业运、职业方向 | 工作场景偏好 |
| 田宅宫 | 房产、居住环境 | 安全感来源 |
| 福德宫 | 精神生活、福气 | 情绪触发模式 |
| 父母宫 | 父母缘分、长辈关系 | 权威关系模式 |

### 对宫关系（180°相对）
命宫↔迁移宫 | 兄弟宫↔交友宫 | 夫妻宫↔官禄宫 | 子女宫↔田宅宫 | 财帛宫↔福德宫 | 疾厄宫↔父母宫

### 三合关系（120°）
- 第一组：命宫+财帛+官禄
- 第二组：兄弟+疾厄+田宅
- 第三组：夫妻+迁移+福德
- 第四组：子女+交友+父母

## 四、核心金句

## 五、排盘算法要点

根据天纪第20-21课：
- 阳年男命/阴年女命：大运顺排
- 阴年男命/阳年女命：大运逆排
- 起运时间：3天=1岁，1天=4个月
- 排八字以北斗星君为主
