**Generate your AI collaboration profile** — a structured text that captures your communication preferences and work style. Paste it into any AI tool (ChatGPT, Claude, Cursor, Hermes…) and it immediately knows how to work with you.

---

> Current interface language: Chinese only. English localization is planned but is not included in v0.7.2.

## Quick Start

1. Open **[diandian1001.github.io/ai-warmstart](https://diandian1001.github.io/ai-warmstart/)**
2. Choose **Professional Mode** (default, recommended) for a clean AI collaboration profile
3. Pick your primary scenario, answer ~9 preference questions (~3 min)
4. Copy the generated text → paste into your AI tool's system prompt

> 🎭 **Experimental Mode** includes optional MBTI type and Zi Wei Dou Shu (Chinese astrology) for entertainment and self-exploration. It is clearly labeled and does not affect Professional Mode output.

---

## Example Output

### What you answer (Professional Mode, Research scenario):
- Language: Chinese + English for technical terms
- Feedback: Point out problems directly
- Response length: Moderate
- Task focus: Help me prioritize
- AI role: Thinking partner
- Learning style: Problem-driven
- Research bottleneck: Too much info, can't tell what's credible
- Output format: Structured report

### What you get (ready to paste into any AI system prompt):

```
# AI Profile · Research & Learning

## General Preferences
- Language: Chinese + English
- Feedback Style: Direct correction
- Response Length: Moderate
- Task Focus: Priority assistant
- AI Role: Partner

## Learning Style
Problem-driven

## Research Bottleneck
Info overload

## Output Format
Structured report

## AI Help
Source collection
---
Generated by Warmstart v0.7.0
```

→ Paste into ChatGPT Custom Instructions, Claude Project Knowledge, Cursor `.cursorrules`, or Hermes `SOUL.md`.

## What It Collects

| Category | Questions | Purpose |
|----------|-----------|---------|
| Scenario | 1 | Determines which preference dimensions to ask |
| General preferences | 5 | Language, feedback style, reply length, task focus, AI role |
| Scenario-specific | 4 | Writing style / tech level / research approach etc. (varies by scenario) |

**Professional Mode** (default): Only the above — no personal data required.

**Experimental Mode** (opt-in): Additionally offers optional MBTI type and birth info for Zi Wei Dou Shu astrology. These are clearly labeled as entertainment-only and do not affect the professional profile dimensions.

## Supported AI Tools

Warmstart generates plain text, not tied to any platform:

| Tool | Paste Into |
|------|-----------|
| ChatGPT | Custom Instructions |
| Claude | System Prompt or Project Knowledge |
| Cursor / Windsurf | `.cursorrules` / `.windsurfrules` |
| Hermes | `SOUL.md` |
| Any AI | Conversation opener or system prompt |

## Privacy

**All data is processed entirely in your browser.** Nothing is uploaded to any server. The current version provides a Chinese-only interface and does not use localStorage, sessionStorage, cookies, analytics, or third-party APIs. Questionnaire answers, birth information, MBTI type, and generated profiles are held only in page memory and disappear when the page is refreshed or closed.

See [PRIVACY.md](PRIVACY.md) for full details.
